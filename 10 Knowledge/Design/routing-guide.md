# Routing Guide

Use this guide before dispatching any design work. The human answers the
questions and selects the Rune.

## Questions
1. Does the brief want typography to feel physical, direct, and deliberately
   structured?
2. Is a light-first composition required?
3. Are 2px borders and zero radius acceptable as core identity rules?
4. Should orange act as a structural element rather than decoration?
5. Does the product benefit from a precise, disciplined, non-generic tone?
6. Is the output intended to feel like Press, not a soft SaaS default?

## Result
- If the answer set matches Press, select `Press`.
- If the brief conflicts with Press, stop and wait for another Rune to be
  authored.

## Notes
- Routing is human, not agentic.
- Do not invent fallback identities.
