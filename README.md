# Sanctum

Sanctum is the compiled knowledge wiki for the Sanctum + Bifrost system.

It stores immutable sources in `40 Sources/` and authored synthesis everywhere
else. The repo is private and is maintained by humans and agents under the
promote gate.

Current Phase 0 contents:
- `01 Indexes/`
- `10 Knowledge/Design/`
- `40 Sources/Design/sessions/`
- `99 Templates/`

The canonical system spec for the split lives at `system-spec.md`, mirrored from
the Bifrost repo.
